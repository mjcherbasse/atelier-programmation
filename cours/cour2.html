HTML: Liens et images

Lien vers un autre page dans notre site

Quand nous avons plusieurs pages HTML dans notre site, on peux crée des liens entré eux en utilisant le sytax <a href="cour1.html">ici</a>
Et dans un vrai HTML ici
Rapelle: Dans l'attribut href il faut avoir le chemin complet pour arriver au fichier depuis l'emplacement de notre html. Dans ce site cour1.html, et cour2.html sont dans le même répertoire.

Images

Pour rajouter une image on utilise la balise img, par example:
<img src="../images/image1.jpg" title="Chouquette" alt="Moi et chouquette" width="200px;">
Comme pour le lien, l'attribut src indique le chemin complet vers le fichier d'image. La notation .. indique un répertoire en arriere.

Comment trouver des images?

Soit chez vous, utilisez vos images. Soit sur Internet, mais en cherchant des images avec licences CC Creative Commons. Par example sur: https://www.flickr.com/creativecommons/

CSS: pour des pages plus stylés!

Pour enrechir le style de notre page, on utilise la language CSS. Pour cela nous rajoutons un block style dans la partie head de notre page html
<style>
..... ici on mets le code CSS
</style>
Dans ce block on va inserer nos code CSS qui va avoir le form de:
selector : {
  css property: value;
  css property: value;
}
Le "selector", nous permet de choisir un element dans notre page sur lequel on va appliquer notre style. Nous avons appris à utiliser les balises HTML comme selectors, par example h1, ou body, mais il y a aussi des autres selector. .
Les css property c'est le propriété visuel que nous volons modifier, ça pourait être coleur, taille, coleur de fond, type de polices, et beaucoup des autres.
Le "value" indique la valeur que nous volons appliquer. Le CSS properties et values, sont défini par W3C et font partie de la définition de CSS, nous ne povonus rien invinter!
Dans ce deuxième cours, nous avons appris les properties:
  • color: Pour modifier la coleur de texte. Par example "color:red". Une liste de coleurs possible se trouve facilement sur Internet, par example sur le site de w3schools
  • background-color: Pour changer le coleur de fond, même valeur possibles que pour color
  • background-image: Pour mettre une image dans le fond. Dans ce cas le valeur est url('chemin_vers_fichier_image')
Nous avons donc pu ajouter à notre page, une partie style pour modifier les coleurs:
<style>
h1 {
 color:brown;
 background-color: cornsilk;
}
h2 {
 background-color: brown;
 color:cornsilk;
}
body {
 background-image: url('../images/backgrounds/Elegant_Background-5.jpg');
 color: cornsilk;
}
</style>
