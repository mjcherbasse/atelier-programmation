Plus de sélecteurs CSS

Nous avons vu que nous pouvons utiliser les balises HTML comme sélecteur CSS, mais il y en a des autres.A chaque élément HTML nous pouvons ajouter un attribut "class", et/ou un attribut "id"

Attribut class

Permet de définir une classe pour l'élément. C'est nous qui donne le nom de la "class", et nous pouvons donner la même classe à plusieurs élément HTML different. En plus un élément HTML peut avoir plusieurs classes.
Example: <div class="btn btn-success"></div>
Pour utiliser la classe comme sélecteur CSS on mets un . avant le nom, dans notre example ça sera .btn et .btn-success

Attribut ID

Chaque element HTML peut avoir un et un seul ID, cet ID doit être unique dans la page. Example: <div id="id_bouton_pour_exploser"></div>
Pour utiliser le ID comme sélecteur CSS on mets un # avant le nom, dans notre exemple ça sera #id_bouton_pour_exploser

DIV SPAN avec class

<div class="container"> <span class="my_block red">RED ZONE</span> <span class="my_block blue">BLUE ZONE</span> <span class="my_block yellow">YELLOW ZONE</span> </div> <style> .red {background-color: red;} .blue {background-color: blue;} .yellow {background-color: yellow;} .my_block { width: 30%; display: inline-block; } .container { width: 100%; } </style>
RED ZONE BLUE ZONE YELLOW ZONE
Esayons autres choses pour le style de ces class
  • Utilisation de pourcentage pour le "width"
  • SPAN avec "display:inline-block;"
  • padding{padding: 20px;}
  • margin{margin: 20px;}
  • borders {border: 5px solid red;border-radius: 16px;}

Links stylés

Quatre "selectors" pour modifier le style de liens
  • a:link - lien normal jamais visité
  • a:visited - lien normal déjà visité
  • a:hover - lien avec la souris
  • a:active - lien pendnat le "click"
