Presentation de l'atelier

Les ateliers programmation à la MJC de l'herbasse, sont destiné aux collégiens. Ils sont étalés sur plusieurs samedi après midi. Le premiers a été le samedi 10/12. Le deuxième sera le samedi 21/1/2017. Ces atelier vont apprendre la base de la programmation avec HTML/CSS/JavaScript A la fin les enfants vont pouvoir crée leur propre page web dynamique
